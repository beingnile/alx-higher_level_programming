The concepts covered include:
* Variables
* Data Types
* Operators
* Operator precedence
* Controlling program flow
* Functions
* Objects and Arrays
* Intrinsic Objects
* Module patterns
* var, let and const

> All the files are intepreted on Ubuntu 20.04 LTS using `node`
